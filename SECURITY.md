# Security Policy

## Supported Versions

The following versions of `network_api_sdk` are currently supported with security updates:

| Version | Supported |
|--------|----------|
| 0.1.x  | ✅ Yes   |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately by contacting the maintainer.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Possible impact
- Suggested fix (if available)

---

## Response Time

We aim to:

- Acknowledge reports within 48 hours
- Provide a fix or mitigation as soon as possible

---

## Responsible Disclosure

We kindly ask that you:

- Do not publicly disclose the vulnerability before it is fixed
- Allow time for resolution before publishing details

---

## Scope

Security issues may include (but are not limited to):

- Authentication or token handling vulnerabilities
- Data leakage
- Unsafe request handling
- Improper caching or storage
